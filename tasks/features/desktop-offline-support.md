## 错误分类

| 错误类型 | 是否可重试 | 处理策略 |
|----------|------------|----------|
| 网络超时 | 是 | 自动重试 |
| DNS 解析失败 | 是 | 重试+切换 DNS |
| 404 Not Found | 否 | 立即失败 |
| 403 Forbidden | 否 | 提示换 Cookie |
| 磁盘满 | 否 | 立即失败+清理提示 |
| YouTube 限制 | 是 | 等待+重试 |
