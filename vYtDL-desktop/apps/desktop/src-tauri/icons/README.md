# Icons

Place your application icons here:

- `32x32.png` - 32x32 PNG icon
- `128x128.png` - 128x128 PNG icon
- `128x128@2x.png` - 256x256 PNG icon (Retina)
- `icon.icns` - macOS icon set
- `icon.ico` - Windows icon

You can generate these from a single high-resolution image using:
- https://tauri.app/v1/guides/features/icons
- `npm run tauri icon /path/to/icon.png`
