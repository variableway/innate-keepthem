## Project Components

- **vYtDL** - Go CLI for downloading YouTube videos and playlists
- **vYtDL Desktop** - Cross-platform desktop app (Tauri v2 + Next.js + React 19)
- **URL Extractor** - Chrome extension for extracting video URLs from YouTube pages
