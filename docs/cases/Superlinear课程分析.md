## 二、从视频内容反推课程结构

### 2.1 课程核心理念

视频全文围绕一个核心框架：**名词 vs 动词**。据此推测课程的核心方法论：

```
课程定位：不教"名词"（工具/概念），教"动词"（操作能力）

知识层（名词）                   能力层（动词）
─────────────────               ─────────────────
RAG 概念                        上下文管理、切分、判断
Agent 概念                       评估、诊断、编排
LangChain/MCP 概念               任务拆解、人机协同
Fine-tune 概念                   结果控制、路由、沉淀

课程承诺：5分钟能学会名词 → 但"动词"才是真正值钱的
```

## 三、已确认的教学方法论

从社区定位和视频内容，可以确认以下几点：

### 3.1 "施工围观"模式

> "最快的成长方式不是独自摸索，而是围观高手的施工现场"

这是 Superlinear 的核心教学方法——不是讲理论，而是展示真实项目从头到尾的完整过程，包括：
- 高手的思考过程（为什么这样做）
- 中间遇到的坑和如何解决
- 决策的 trade-off

### 3.2 案例驱动

社区沉淀了 **300+ 真实案例**，涉及：
- AI 编程（Claude Code/Codex CLI 实战）
- AI 写作与内容生产
- AI 知识库/RAG 系统搭建
- AI 工作流自动化
- Agent 开发

### 3.3 "手艺"隐喻

视频反复强调 AI 能力是一门"手艺"（craft），类比高尔夫、拳击：
- 不能通过看理论学会——必须练
- 需要反馈循环——做→看结果→调整
- 高手和新手的差距在细节动作，不在知识量

## 六、初步判断

### 课程的差异化价值

基于现有信息，Superlinear Academy 的核心差异化在于：

1. **框架独特**：名词 vs 动词的二分法是一个清晰的认知框架，比市面上"学 AI 要动手"的泛泛之谈更有穿透力
2. **案例真实**：300+ 真实案例 + "围观施工现场" 的方法，比纯理论课程更有实操价值
3. **社区活跃**：3,000+ 构建者的社区意味着 peer learning 和案例共享
4. **紧跟前沿**：12 期迭代、30% 内容随模型更新，说明课程不是"录制一次卖三年"

### 潜在局限

1. **内容不公开**：课程价值高度依赖"围墙花园"内的内容质量，外人无法评估
2. **个人 IP 依赖**：课程以"鸭哥"个人 IP 为核心，规模化受限于个人精力
3. **定价不透明**：bundle 折扣码的存在暗示价格可能较高或有复杂定价结构
4. **效果难以验证**："动词"能力不像"名词"那样可以在简历上量化，学习效果难以向雇主证明——课程自己也承认这一点

### 与先前调研的关联

本次分析直接验证了"AI时代动词-自主调研报告"中的核心判断：
- Superlinear 的"六动词"框架（上下文管理、切分、判断、评估、诊断、编排）与我们的独立调研高度吻合
- 课程确实在教授"动词"——但具体内容被付费墙保护
- 社区模式（案例驱动 + 围观高手）正是报告中"Level 3 组织赋能者"的一种实现方式
